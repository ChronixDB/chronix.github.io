Quick Start

This quick start example uses the chronix/server docker container and an example project using the Chronix client. You will need at least 15 minutes to be up and running. You can follow the description or watch the video. Now lets start...

(1) Get the and start the docker container with the Chronix server by typing 

docker run -d -p 8983:8983 chronix/server

It will took a while, but then you can query the server using the Chronix client.

(2) Meanwhile we can checkout the Chronix Server Demo from our GitHub Repository 

git clone https://github.com/qaware/chronix.git

In the directory

gradle build
