Chronix Stack

The whole Chronix Stack is open source and is free to use for everyone without any restrictions. One can use the stack using plain Chronix blocks, such as the Chronix Collector, a tool for collecting jmx values via ssh from several applications, or the Chronix Analytics, a JavaFX rich client for time series exploration. If preferred a stack using collectd or logstash as collection framework amd Grafana or Apache Zeppelin is also possible. The following figure shows the Chronix Stack.
