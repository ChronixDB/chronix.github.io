Quick Start

Get up and running with Chronix in 10 minutes with our quickstart guide:.

1. Prerequisites

You need a Java 8 installed on your system. That's it :-)

2. Download

The Chronix showcase consists of two parts. The Chronix Server (download) and an example JavaFX application (download) for time series exploration (pick the latest release, respectively). Unzip the Chronix Server, the JavaFX part is an executable Java archive.

3. Run

The provided Chronix Server contains one week of operational time series data. Thus we can start the server and everything is up and running. Let's open a terminal and type the following commands to download and start the Chronix Server:

Chronix is now running on localhost:8983. Now we can start the JavaFX application. Download the executable jar from GitHub and start it from the console.

chronix@chronixDB:~$

With the running JavaFX application, we can query Chronix for time series data. A simple query that delivers all time series whose metric name contains *Load* looks like this: 

metric:*Load*

Just enter the query term into the text box and hit Shift+Enter to retrieve the result:


The following query returns the maximum value for the *Load* query: 

ag=max


These examples represent only two simple queries, check out our examples repository for more details.
