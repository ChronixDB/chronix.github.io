Quick Start

The quickstart guide show how to run and play with Chronix within 10 minutes from now.

1. Prerequisites

To run the Chronix showcase you need Java 8 installed on your system. That's all :-)

2. Download the Chronix Showcase

The Chronix showcase consists of two parts. The Chronix Server and an example JavaFX application for time series exploration. Both can be downloaded from the GitHub repository (take the latest release).

  1. Chronix Server: Download
  2. Chronix JavaFX Showcase: Download

The Chronix Server is a zip archive that have to be extracted. The Chronix JavaFX is an executable java archive.

3. Run the Chronix Showcase

Chronix Server contains one week of operational time series data. Thus we can start the server and everything is up and running. Let's open a terminal and type the following commands to download and start the Chronix Server:

chronix@chronixDB:~$

Chronix is now running on localhost:8983. Now we can start the JavaFX application. Download the executable jar from GitHub and start it from the console.

chronix@chronixDB:~$

If everything works the JavaFX application starts and we can ask Chronix for time series data. A simple query that delivers all time series that metric contains *Load* is: 

metric:*Load*

To get the maximum value for the *Load* query just enter the following filter query 

ag=max

That are only two simple queries. For more details check out our GitHub Examples Repository.
